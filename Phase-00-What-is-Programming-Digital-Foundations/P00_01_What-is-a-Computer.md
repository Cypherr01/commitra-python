# Topic: What is a Computer?

## What Is This?
A computer is a machine that can process information and perform tasks. It's a complex system made up of various components that work together to enable us to interact with it.

## How It Works Internally
Imagine a computer as a brain that can understand and follow instructions. It uses a combination of the following components to perform tasks:

- **CPU (Central Processing Unit)**: The brain that executes instructions.
- **RAM (Random Access Memory)**: Temporary fast memory that holds data while the computer is turned on.
- **Storage**: Permanent slow memory that stores data even when the computer is turned off (e.g., hard drive or solid-state drive).
- **I/O devices**: Input/Output devices like keyboard, screen, mouse, and network that allow us to interact with the computer.

## Common Mistakes Beginners Make

### 1. Misunderstanding the CPU's role
Example: Thinking the CPU is a storage device because it stores data temporarily. **Incorrect**.
The CPU executes instructions, not stores data.

### 2. Confusing RAM and Storage
Example: Believing that RAM is permanent storage because it's fast and holds data. **Incorrect**.
RAM is temporary fast memory that loses its data when the computer is turned off.

### 3. Overlooking I/O devices
Example: Assuming that the computer can interact with the world without I/O devices like keyboard and screen. **Incorrect**.
I/O devices are essential for interacting with the computer.
